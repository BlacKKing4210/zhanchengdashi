# Rank Castle Assets

Generated for this project on 2026-07-18 with the OpenAI image generation tool, using the existing `assets/art/buildings/base.png` only as the style and silhouette reference.

The source sheet used a 4x2 grid in this order:

1. Bronze
2. Silver
3. Gold
4. Platinum
5. Diamond
6. Star
7. King
8. Empty

The runtime PNGs were deterministically split, chroma-keyed, edge-cleaned, and normalized to transparent 512x512 canvases. No animal artwork was changed.
