# 新版召唤动物与流派设计案

生成日期：2026-07-06

用途：基于当前 60 张动物卡，提出下一版召唤动物的玩法设计。本文只作为审阅策划案，不修改配置表，不进入游戏实现。

## 1. 设计目标

当前战斗里，动物主要差异来自攻击、生命、速度、射程和少量技能描述，玩家实际观感容易变成“地块解锁后又出一只动物”。新版设计要把动物拆成更清楚的战场职责，让每次翻出动物营地时都能形成阵容期待、克制判断和局内选择。

核心目标：

- 让动物不只是数值不同，而是召唤形态不同：轻型以双单位为主，重型单体压线，远程需要保护，辅助放大组合价值。
- 先建立几条大流派，再给动物找位置，避免 60 张卡各自孤立。
- 每条流派都要有明显优点、明显短板和可被识别的克制关系。
- 动物之间要有“组合句法”：例如蚂蚁带群体，山羊扶前排，猎鹰点后排，袋鼠切远程。
- 每种战场类型和关键反制动物都要有清楚的克制关系，但这些关系作为策划和平衡依据，不直接写进卡牌或战斗 UI。
- 大幅降低低中品质技能密度：绝大部分绿色、大部分蓝色、小部分紫色小动物可以没有技能，只靠召唤数量、属性、速度、射程和目标偏好形成差异。
- 有技能的动物也以 1 个清楚特性为主，避免一张卡同时拥有战吼、亡语、光环、成长等多个规则。
- 仍兼容当前 8 格编组结构：金矿卡和至少 1 张防御塔卡占位后，动物位默认按 6 张设计。

## 2. 当前问题

| 问题         | 影响                       | 新版方向                       |
| ---------- | ------------------------ | -------------------------- |
| 多数动物都是单个单位 | 战斗读起来像数值互殴               | 按类型定义召唤数量、站位和目标优先级         |
| 动物定位散      | 玩家难以围绕卡组做取舍              | 先定义大流派，再把动物放入流派            |
| 克制关系不明显    | 买地块时缺少判断                 | 用群体、重型、远程、突击、辅助、运营形成克制链    |
| 辅助价值不直观    | 辅助容易像低面板卡                | 让辅助明确绑定一类队友或一条流派           |
| 高品质只是更强    | 卡牌成长缺少风格升级               | 高品质动物提供流派核心、反制点或翻盘点        |
| 技能描述过密     | 玩家记不住每只动物的独立规则，实现和表现成本也高 | 用“无技能属性单位 + 少量单特性单位”构成卡池骨架 |

## 3. 战场类型

### 3.1 类型总览

| 类型  | 召唤表现           | 战场价值            | 明显弱点             | 代表动物            |
| --- | -------------- | --------------- | ---------------- | --------------- |
| 普通  | 1 只，属性均衡       | 稳定补线、承接前后排      | 没有极端优势，容易被专门克制   | 猫、鹿、狼、狮子        |
| 轻型  | 通常 2 只，紫色最多 3 只，单体生命非常低 | 抢线、围攻、压远程和辅助    | 怕范围伤害、反伤、持续减速    | 老鼠、蚂蚁、麻雀、蝌蚪、鸭   |
| 重型  | 1 只，生命很高，速度慢   | 抗线、保护后排、拖住突击    | 怕远程集火、毒、控制、经济绕线  | 蜗牛、乌龟、河马、大象、猛犸象 |
| 远程  | 1 只，射程长，生命明显偏低 | 安全输出、压重型、点杀关键目标 | 怕轻型贴脸、突击切入和远程反远程 | 孔雀、猎鹰、老鹰、金雕     |
| 突击  | 1-2 只，速度快或带跳击  | 切后排、打断辅助、惩罚贪经济  | 怕重型堵路、眩晕、护盾阵     | 狐狸、袋鼠、猞猁、老虎、虎鲸  |
| 辅助  | 1 只，低单兵价值      | 护盾、治疗、光环、复制、经济  | 怕被突击点掉，怕阵容人数不足   | 山羊、羊、水獭、天鹅、蓝鲸   |
| 运营  | 1-2 只，战斗力较低    | 产金币、修复、占点收益、拖后期 | 怕早期快攻和后排切入       | 仓鼠、松鼠、猪、牛、河狸    |

### 3.3 小数值属性框架

下一版数值先整体收窄，避免动物差异只靠大数字堆叠。卡牌价值主要来自召唤数量、触发时机、目标选择、攻击形状和配合关系。

| 对象      | 绿色参考值      | 设计说明                           |
| ------- | ----------:| ------------------------------ |
| 无技能绿色动物 | 1 攻 / 3 血  | 普通单体的基础参考，技能动物和多单位动物要为机制付出面板折价 |
| 轻型小动物   | 1 攻 / 1 血  | 单体非常脆，通常 2 只一组；紫色轻型最多 3 只        |
| 绿色防御塔   | 1 攻 / 10 血 | 稳定点射，主要克制快攻拆建筑和低血突击            |
| 基地      | 2 攻 / 30 血 | 有基础自卫能力，但不能替代防御塔               |
| 绿色营地建筑  | 8 血        | 早期可被快攻威胁，促使玩家保护关键营地            |

速度只保留三档：快、中、慢。快用于轻型、突击和拆建；中用于普通、远程和多数辅助；慢用于重型和终局单位。

### 3.4 克制与身板规则

克制关系优先通过基础参数表达：召唤数量、生命预算、速度档、射程档、目标优先级和攻击形状。只有当规则会改变战斗结算时，才写成技能。

同品质普通近战建议生命基准：绿色 3、蓝色 5、紫色 8、金色 12。远程生命约为同品质普通近战的 1/2，超远程生命约为 1/3，最终落表时向上取整。

| 定位    | 生命预算      | 主要补偿         | 克制对象         | 被克制对象           |
| ----- | --------- | ------------ | ------------ | --------------- |
| 普通近战  | 100%      | 稳定攻击和站线      | 轻型、低血突击      | 反重型、远程集火、AOE    |
| 轻型多单位 | 25%-45%   | 双单位、速度、触发次数；紫色最多 3 只 | 远程、辅助、单点慢攻   | AOE、反伤、击杀成长     |
| 重型    | 170%-280% | 高血、保护后排      | 突击、单点爆发、低攻轻型 | 反高生命、毒、远程集火     |
| 远程    | 约 50%     | 射程、安全输出、目标偏好 | 重型、慢速前排、低速控制 | 快速突击、轻型贴脸、远程反远程 |
| 超远程   | 约 33%     | 最远距离、后排点杀    | 辅助、远程核心、运营后排 | 狐狸/袋鼠/猞猁等切后排单位  |
| 突击    | 65%-105%  | 快速接敌、优先打后排   | 远程、辅助、运营建筑   | 重型堵路、防御塔、眩晕     |
| 辅助/运营 | 60%-130%  | 光环、护盾、治疗、资源  | 消耗战、铁壁、数量流   | 突击、远程点杀、快攻拆建    |

关键反制模板：

| 模板      | 基础规则                    | 代表动物          | 主要克制          | 主要被克制           |
| ------- | ----------------------- | ------------- | ------------- | --------------- |
| 快速反远程   | 速度快，目标优先级为远程/辅助/建筑      | 狐狸、袋鼠、猞猁      | 孔雀、猎鹰、长颈鹿、金雕  | 蜗牛、乌龟、河马、防御塔、眩晕 |
| 远程反远程   | 自己是远程，目标优先级为敌方远程        | 猎鹰、金雕         | 对方远程、超远程、辅助后排 | 快速反远程、轻型贴脸      |
| 反高生命    | 对最大生命或当前生命较高的目标额外伤害/毒压血 | 鳄鱼、科莫多龙       | 蜗牛、河马、大象、北极熊  | 老鼠、蝌蚪、麻雀等轻型多单位  |
| 反轻型 AOE | 攻击形状为顺劈、震击、散射或弹射        | 熊、大猩猩         | 老鼠、蚂蚁、蝌蚪、鸭、斑马 | 河马、大象、北极熊等高血单体  |
| 拆建快攻    | 目标优先级为金矿/营地/防御塔         | 麻雀、狐狸、松鼠、鹿、豹子 | 运营、远程阵地、贪经济   | 防御塔、重型、控制       |
| 控制反突击   | 首次接敌眩晕、缠绕或护盾挡线          | 野猪、犀牛、巨蟒、熊    | 狐狸、袋鼠、老虎、虎鲸   | 反高生命、远程集火       |

目标偏好可以属于无技能参数。例如“优先攻击远程”“优先攻击建筑”“优先攻击高生命”不一定写成技能；只有“对高生命 +1 伤”“击杀后成长”“入场标记”等额外结算才算单特性。克制关系不作为卡面关键词展示，玩家主要通过动物行为、目标选择、攻击形状和战斗结果理解克制。

### 3.5 触发关键词

关键词只用于少数有技能动物，不是每张卡都必须拥有。下一版默认把“无技能但属性/召唤形态鲜明”的动物视为正常卡牌，而不是过渡卡。

一只动物原则上只拥有 1 个主要特性。目标偏好、召唤数量、速度档、射程和基础属性属于基础参数，不计入技能；战吼、亡语、友亡、击杀成长、光环、顺劈/散射等改变战斗规则的效果计入技能。

| 关键词   | 触发时机            | 设计用途                     | 代表动物             |
| ----- | --------------- | ------------------------ | ---------------- |
| 无技能   | 不触发额外规则         | 用更清楚的属性、数量、射程或速度承担战场职责   | 蜗牛、猫、海豹、河马、豹子、鲨鱼 |
| 战吼    | 动物入场时触发一次       | 让翻开营地立刻有反馈，适合护盾、冲锋、震慑、标记 | 羊、鹿、野猪、熊、犀牛、猛犸象  |
| 亡语    | 动物死亡时触发一次       | 让死亡成为资源，支持多单位和献祭玩法       | 鸽子、鸭、猪、狮子        |
| 友亡    | 任意友方动物死亡时触发     | 把多单位和死亡流连接起来，形成越死越强的局面   | 银背猩猩             |
| 击杀    | 该动物击杀敌方动物或建筑时触发 | 反制多单位，小单位会喂收割型猎手         | 老鹰、虎鲸            |
| 群势    | 根据场上友方动物数量计算    | 让多单位和光环流形成正反馈，但怕 AOE 清场  | 蚂蚁、企鹅、斑马、山羊、大象   |
| 顺劈/震击 | 近战攻击同时命中主目标附近敌人 | 近战 AOE，用来克制轻型群体          | 熊、大猩猩            |
| 散射/弹射 | 远程攻击命中多个目标      | 作为后续远程 AOE 预留，首版谨慎投放     | 后续候选：孔雀、海豚、金雕    |
| 拆建    | 优先攻击建筑          | 快攻流的明确目标，专门威胁营地、金矿和防御塔   | 麻雀、狐狸、松鼠、鹿、豹子    |

### 3.6 品质与技能复杂度预算

下一版卡池先控制技能密度，再用属性、召唤数量和职责组合做差异。这样玩家能先理解“这是什么类型的动物”，再慢慢认识少数核心技能。

| 品质  | 技能密度建议        | 设计原则                                                    |
| --- | ------------- | ------------------------------------------------------- |
| 绿色  | 约 75%-85% 无技能 | 主要负责新手识别：轻型、普通、重型、远程、突击、辅助的基础体验。少数绿色技能只做一个很直观的小特性。      |
| 蓝色  | 约 55%-70% 无技能 | 作为进阶属性和阵容拼图。大部分蓝色仍靠面板、速度、射程、召唤数和目标偏好；少数蓝色提供单特性光环、亡语或战吼。 |
| 紫色  | 约 20%-40% 无技能 | 紫色小动物可以没有技能，用更极端的召唤形态或属性成为高级拼图；有技能的紫色也尽量只保留一个战场特性。      |
| 金色  | 约 0%-20% 无技能  | 主要承担终局核心，但仍允许少数“纯面板传说”作为简单直接的强卡。                        |

技能复杂度限制：

- 绿色技能只允许非常短的规则：例如“死亡返 1 金”“附近轻型 +1 攻”“入场给 1 点护盾”。
- 蓝色技能可以成为流派拼图，但仍只做单一特性，例如“死亡返金”“入场冲锋”“附近友军小护盾”。
- 紫色技能可以是流派核心，但不叠第二触发；例如有亡语就不再有友亡成长，有顺劈就不再有击杀成长。
- 金色技能可以更强，但仍尽量一张卡一个记忆点。只有未来首领或特殊活动卡才考虑双特性。

## 4. 大流派

### 4.1 鼠潮快攻

核心体验：低费动物以双单位为主，快速铺线、抢格、压远程。玩家看到小单位连续冲出去时，能立刻感到“战线被推开了”，但不会因为一次召唤 5-8 只导致画面和结算失控。

优势：

- 开局强，能快速扩大地块接触面。
- 对远程和辅助有天然压力，靠数量分散单点伤害。
- 容易触发光环、护盾、阵亡效果。

弱点：

- 极怕范围伤害、反伤、减速和高频防御塔。
- 单体生命低，遇到重型加范围控制会很难穿透。
- 后期如果没有光环或复制，单位会快速蒸发。

核心动物：老鼠、蚂蚁、麻雀、鸽子、蝌蚪、鸭、松鼠、狐狸、马、斑马、豹子。

推荐 6 动物位：老鼠、蚂蚁、蝌蚪、鸭、松鼠、斑马。

明显克制：远程火力流、金果运营流。

明显被克制：铁壁堡垒流、范围控制、反伤防线。

### 4.2 铁壁堡垒

核心体验：重型动物慢速推进，像一堵墙向前压。玩家的主要决策是用金币买出连续前排，给后排和金矿争取时间。

优势：

- 抗突击能力强，能保护后排和辅助。
- 对单点爆发不敏感，能拖到高品质动物进场。
- 适合和护盾、治疗、生命光环组合。

弱点：

- 移速慢，容易被远程风筝和毒控磨掉。
- 如果没有后排，容易只是在原地挨打。
- 害怕经济被快攻切断。

核心动物：兔子、蜗牛、狗、乌龟、猪、海豹、牛、骆驼、熊、犀牛、河马、大象、北极熊、猛犸象。

推荐 6 动物位：蜗牛、乌龟、海豹、熊、山羊、羊。

明显克制：獠牙突击流、单点爆发流。

明显被克制：鹰眼远火流、毒控消耗、运营绕线。

### 4.3 鹰眼远火

核心体验：前排拖住敌人，远程动物在后方持续点杀。远程不应该只是一条更长的攻击线，而要承担“打谁”和“怎么打”的差异。

优势：

- 压重型效率高，能稳定消耗慢速前排。
- 适合点杀辅助、标记高价值目标。
- 防守反击强，配合塔能打出安全区。

弱点：

- 怕轻型贴脸，远程容易被小单位淹没。
- 怕突击绕后，尤其是袋鼠、猞猁、老虎。
- 没前排时很脆。

核心动物：青蛙、鸡、孔雀、海豚、猎鹰、长颈鹿、老鹰、金雕、巨蟒。

推荐 6 动物位：兔子、孔雀、海豚、猎鹰、老鹰、金雕。

明显克制：铁壁堡垒流、慢速控制流。

明显被克制：鼠潮快攻流、獠牙突击流。

### 4.4 獠牙突击

核心体验：高速动物绕开正面或短时间爆发，专门惩罚远程、辅助和贪经济阵容。

优势：

- 对后排威胁清晰，能让远程阵容必须带前排。
- 节奏快，适合抢中线和打断金矿收益。
- 高品质突击动物能制造翻盘点。

弱点：

- 正面打重型效率低。
- 怕眩晕、护盾、减速和反伤。
- 如果突击目标选择错误，容易白白送掉。

核心动物：猫、狐狸、鹿、袋鼠、野猪、猞猁、老虎、豹子、虎鲸、鲨鱼。

推荐 6 动物位：狐狸、鹿、袋鼠、野猪、猞猁、老虎。

明显克制：鹰眼远火流、牧歌续航流、金果运营流。

明显被克制：铁壁堡垒流、控制防线。

### 4.5 牧歌续航

核心体验：辅助动物让前排更厚、群体更持久、远程更安全。该流派不是靠单卡强，而是靠“越打越不亏”。

优势：

- 持续战斗强，能把普通动物和重型动物放大。
- 防守稳定，适合拖到高品质卡牌和金矿收益。
- 对轻型群体有保护价值，能延长小单位存活。

弱点：

- 单独作战很弱，辅助被切掉会崩。
- 怕爆发和斩杀，护盾/治疗来不及时会亏。
- 需要站位和前排，不能裸上。

核心动物：山羊、羊、猴子、水獭、企鹅、天鹅、鹤、长颈鹿、蓝鲸、猛犸象。

推荐 6 动物位：山羊、羊、水獭、天鹅、鹤、蓝鲸。

明显克制：普通均衡流、消耗战。

明显被克制：獠牙突击流、斩杀点杀。

### 4.6 金果运营

核心体验：用经济动物和建筑互动建立长期优势。运营流不应该是“躲着发育”，而是用战线选择保护收益点。

优势：

- 中后期资源优势明显，能买更多地块和高价营地。
- 和金矿、防御塔、修复类效果有天然关联。
- 能用地块数和建筑分数赢倒计时。

弱点：

- 前期战力弱，怕快攻直接压进来。
- 关键运营动物被切掉时损失大。
- 需要防御塔或重型动物保护。

核心动物：仓鼠、鸽子、松鼠、猪、牛、河狸、骆驼、海豚。

推荐 6 动物位：仓鼠、松鼠、猪、牛、河狸、骆驼。

明显克制：慢速防守、无突击的铁壁阵容。

明显被克制：鼠潮快攻流、獠牙突击流。

## 5. 克制关系

| 发起方  | 主要压制     | 压制原因            | 主要怕谁       |
| ---- | -------- | --------------- | ---------- |
| 鼠潮快攻 | 远程、辅助、运营 | 数量多，能分摊单点伤害并贴脸  | 范围、反伤、重型控制 |
| 铁壁堡垒 | 突击、单点爆发  | 高生命和护盾让爆发打不穿    | 远程、毒控、经济绕线 |
| 鹰眼远火 | 重型、慢速前排  | 安全距离持续输出        | 轻型、突击      |
| 獠牙突击 | 远程、辅助、运营 | 高速切入，优先打高价值低血目标 | 重型、眩晕、护盾   |
| 牧歌续航 | 均衡、消耗战   | 护盾治疗让换血变赚       | 爆发、斩杀、切后排  |
| 金果运营 | 慢速防守     | 资源和地块数滚雪球       | 快攻、突击      |

### 5.1 个体克制模板

流派克制之外，每个关键动物还要承担一个清楚的局部反制职责。这样玩家不只是在选“快攻/远程/铁壁”，也能在编组里回答“我怎么处理远程”“我怎么处理重型”“我怎么防多单位”。

| 反制职责  | 典型动物          | 规则表达              | 克制对象          | 被克制对象           |
| ----- | ------------- | ----------------- | ------------- | --------------- |
| 快速切远程 | 狐狸、袋鼠、猞猁      | 快速，远程/辅助目标优先      | 孔雀、猎鹰、长颈鹿、金雕  | 蜗牛、乌龟、河马、防御塔、眩晕 |
| 远程打远程 | 猎鹰、金雕         | 远程单位优先；金雕作为超远程但极脆 | 对方远程、辅助后排     | 狐狸、袋鼠、猞猁、轻型贴脸   |
| 打高生命  | 鳄鱼、科莫多龙、巨蟒    | 高生命目标额外伤害、毒压血或缠绕  | 重型、护盾前排、终局大单位 | 轻型多单位、远程点杀      |
| 清多单位  | 猫、刺猬、熊、大猩猩    | 高攻近战、站线拦截、顺劈/震击   | 老鼠、蚂蚁、蝌蚪、鸭    | 河马、大象、北极熊、反高生命  |
| 击杀成长  | 老鹰、虎鲸         | 击杀低血或残血后获得收益      | 轻型、续航残血       | 重型拖住、控制、反远程突击   |
| 拆建筑   | 麻雀、狐狸、松鼠、鹿、豹子 | 建筑目标优先            | 金矿、远程营地、防御塔缺口 | 防御塔、重型挡线、控制     |
| 保护后排  | 蜗牛、乌龟、狗、海豹、河马 | 高生命慢速前排           | 突击、单点爆发       | 反高生命、毒、远程集火     |

首版原则：每个动物最多承担一个主要反制职责。比如猎鹰做“远程打远程”，就不再附带击杀成长；鳄鱼做“打高生命”，就不再附带眩晕或毒。

### 5.2 死亡流：多单位喂成长

死亡流的核心不是“单位死了也不亏”，而是让死亡成为下一波推进的燃料。轻型动物提供死亡次数，亡语动物提供死亡收益，友亡动物把这些死亡转成攻击、护盾或召唤。

简化原则：大部分死亡素材本身无技能，只负责“数量多、血少、容易死”。真正有亡语或友亡成长的动物只保留少数，否则死亡流会变成满屏触发文字。

| 模块      | 动物          | 作用                            |
| ------- | ----------- | ----------------------------- |
| 无技能死亡素材 | 老鼠、蝌蚪、麻雀、狐狸 | 一次召唤 2 只或速度快，容易死亡，负责提供触发次数和战线压力 |
| 少量亡语收益  | 鸽子、鸭、猪、狮子   | 死亡返金币、补位、召唤新单位或返场，每只只保留一种亡语   |
| 友亡成长核心  | 银背猩猩        | 友方动物死亡达到阈值后获得攻击或生命成长，不再叠加额外亡语 |
| 放大器     | 羊、天鹅、蓝鲸     | 给小单位护盾，让死亡流不至于被瞬间清空           |

建议规则：友亡成长不按每次死亡都立刻加 1，避免小单位过度滚雪球。推荐每 2-3 个友方动物死亡触发一次，且有上限。

### 5.3 光环流：数量越多越强

光环流和多单位天然相配，但必须让玩家看懂“场上动物数量”正在驱动战力。群势类动物不一定自己很强，但当己方动物数量达到阈值时，会让整条线突然变厚或变凶。

简化原则：光环动物数量要少，且每只只提供一种加成。老鼠、蝌蚪、马、豹子等双单位或三单位动物不需要自己带光环，它们只负责把数量阈值撑起来。

| 触发条件    | 动物  | 效果方向               |
| ------- | --- | ------------------ |
| 友方动物 4+ | 蚂蚁  | 轻型动物攻击 +1，强化早期小队   |
| 友方动物 5+ | 企鹅  | 全体攻击小幅提升，适合普通和轻型混编 |
| 不同类型 3+ | 斑马  | 每有一种不同主类型，提高附近友军攻击 |
| 友方动物 6+ | 山羊  | 全体生命上限 +1，适合铁壁和鼠潮  |
| 友方动物 8+ | 大象  | 大范围生命光环，让终局推进成型    |

反制点：AOE 击杀和突击切光环核心。光环动物本体不能太肉，否则数量流会同时拥有爆发和稳定性。

### 5.4 击杀流：反制多单位

击杀流专门惩罚多单位。轻型小动物虽然能触发光环和死亡收益，但也会给收割型动物提供目标。

简化原则：绿色和蓝色优先用无技能高攻/高速动物来反制小单位，不要每只都做击杀成长。真正的击杀成长只给少数紫色或金色核心。

| 动物  | 收割方式               | 克制对象      | 风险       |
| --- | ------------------ | --------- | -------- |
| 猫   | 无技能高攻，靠基础攻击清小单位    | 老鼠、蝌蚪、鸭   | 怕重型挡路    |
| 猞猁  | 无技能高速刺客，优先找低血或远程目标 | 远程、小动物    | 怕眩晕和反伤   |
| 老鹰  | 单特性：优先斩杀低血目标       | 续航残血、多单位  | 怕被突击切掉   |
| 虎鲸  | 单特性：击杀后回复 1 血      | 重伤前排、小单位  | 怕铁壁拖住    |
| 鲨鱼  | 无技能传说高输出，直接突破单点    | 低血群体、脆皮后排 | 怕控制和重型拖住 |

如果未来保留“击杀成长层数”，优先只放在 1-2 张高品质核心动物上，并用攻击旁边的小爪痕数字显示层数。

### 5.5 AOE 流：清理多单位

AOE 是多单位的硬克制。为了避免所有 AOE 都像同一种技能，近战使用顺劈和震击，远程使用散射和弹射。

简化原则：AOE 本身就是一个完整技能，不再额外叠加击杀成长、护盾或亡语。绿色/蓝色 AOE 优先做成小范围、低伤、低记忆负担。

| 类型    | 动物         | 攻击形状               | 设计目的            |
| ----- | ---------- | ------------------ | --------------- |
| 低费拦截  | 刺猬         | 无技能高血普通近战，站住轻型贴脸路线 | 用属性克制小单位，不做额外反伤 |
| 近战冲撞  | 野猪、犀牛      | 首次接敌对一条短线造成伤害/眩晕   | 打乱群体和前排         |
| 近战震击  | 熊、大猩猩      | 单特性范围攻击或震慑         | 中后期清小单位         |
| 远程点杀  | 老鹰、金雕      | 优先低血或最远目标          | 清残血与后排核心        |
| 远程多目标 | 后续候选：孔雀、海豚 | 若需要再做散射/弹射         | 首版不急着给蓝色远程加技能   |

AOE 的短板是对单体重型效率不应过高。顺劈和散射可以清小单位，但打蜗牛、河马、大象时应该慢。

### 5.6 拆建快攻：优先攻击建筑

快攻流需要一个更清楚的胜利目标：不是只追动物，而是优先拆营地、金矿和防御塔。这样快攻能克制运营和远程阵地，也能让玩家意识到“建筑也需要保护”。

简化原则：优先攻击建筑可以作为目标偏好，不一定算技能。麻雀、狐狸、松鼠、豹子可以无技能，只靠速度、数量和建筑目标优先级形成快攻体验；鹿这类少数动物再用“入场冲锋”做单特性。

| 动物  | 拆建定位     | 优先级                |
| --- | -------- | ------------------ |
| 麻雀  | 最早期侦察拆建  | 金矿 > 营地 > 防御塔 > 动物 |
| 狐狸  | 快速绕线拆建筑  | 金矿 > 远程营地 > 辅助动物   |
| 松鼠  | 抢格和偷经济建筑 | 空地连接点 > 金矿 > 营地    |
| 鹿   | 战吼加速冲向建筑 | 防御塔 > 金矿 > 营地      |
| 豹子  | 高品质快攻拆建  | 后排营地 > 金矿 > 防御塔    |

反制点：绿色防御塔 1 攻 / 10 血、重型前排、眩晕和减速。快攻动物如果被迫打重型，会明显亏。

### 5.7 战吼与亡语流

战吼负责“入场即产生选择价值”，亡语负责“死亡后继续影响局势”。两者可以形成一进一出的节奏。

简化原则：战吼和亡语都是强记忆点，一张动物卡只选一个。不要出现“入场加速、死亡返场、还带光环”的多触发动物。

| 流派   | 动物      | 示例效果                  |
| ---- | ------- | --------------------- |
| 战吼护盾 | 羊、猛犸象   | 入场给自己或附近友军 +1/+2 护盾   |
| 战吼冲锋 | 鹿、野猪、犀牛 | 入场短暂加速或首次接敌眩晕         |
| 战吼标记 | 金雕      | 入场标记最远或最低血目标          |
| 战吼复制 | 猴子      | 入场复制一个低费动物，作为蓝色少数技能核心 |
| 亡语补位 | 鸭、狮子    | 死亡后召唤小单位或返场单位         |
| 亡语资源 | 鸽子、猪    | 死亡返金币，和运营/死亡流相连       |

| 獠牙突击 | 狐狸、鹿、袋鼠、野猪、猞猁、老虎  | 大部分靠速度和高攻切后排，少数冲锋/眩晕开口   |
| 牧歌续航 | 山羊、羊、水獭、天鹅、鹤、蓝鲸   | 通过少数护盾和治疗特性赢长期换血，不堆过多光环  |
| 金果运营 | 仓鼠、松鼠、猪、牛、河狸、骆驼   | 无技能前排保护收益点，少数返金/修复特性拖后期  |

## 9. 表现建议

### 9.1 战场可读性

- 轻型动物出现时按扇形或小队列弹出，间隔 0.08-0.15 秒，避免看起来像一团重叠点。
- 重型动物落地时有小震动、尘土圈或短暂压暗阴影，让玩家一眼知道“这是一堵墙”。
- 远程动物默认站在前排后方，不主动贴脸；攻击弹道更清楚，便于玩家理解它在打谁。
- 突击动物需要短促前冲、跳击或残影，目标优先级要明显。
- 辅助动物的光环范围只在短时间或选中时显示，避免常驻视觉噪音。

### 9.2 卡牌可读性

卡牌上建议新增类型小图标：

- 轻型：双点或小队图标。
- 重型：盾或厚血条图标。
- 远程：箭头或准星图标。
- 突击：爪痕或闪电图标。
- 辅助：星环或十字图标。
- 运营：金币叶片或小旗图标。

## 10. 后续落地字段建议

如果评审通过，后续再进入配置和实现时，可以为 `cards.csv` 或派生表增加以下字段：

| 字段                      | 用途                                                                             |
| ----------------------- | ------------------------------------------------------------------------------ |
| `battle_role`           | 主类型：normal / light / heavy / ranged / assault / support / economy              |
| `spawn_count`           | 每次召唤数量                                                                         |
| `base_attack_small`     | 小数值攻击，建议 0-4                                                                   |
| `base_hp_small`         | 小数值生命，建议 1-14，建筑另算                                                             |
| `speed_tier`            | 快 / 中 / 慢                                                                      |
| `range_tier`            | melee / short / ranged / ultra_ranged                                          |
| `hp_budget_class`       | melee_baseline / light_low_hp / ranged_half / ultra_ranged_third / heavy_bonus |
| `unit_hp_factor`        | 单体生命倍率                                                                         |
| `unit_attack_factor`    | 单体攻击倍率                                                                         |
| `has_skill`             | 是否拥有技能；低中品质大量为 false                                                           |
| `skill_complexity`      | none / single_trait / special；首版只使用 none 和 single_trait                        |
| `primary_trait`         | 单一核心特性，例如 deathrattle_gold / aura_light_attack / battlecry_charge              |
| `trigger_timing`        | none / battlecry / deathrattle / ally_death / kill / interval                  |
| `battlecry_effect`      | 入场效果，例如护盾、冲锋、标记、复制                                                             |
| `deathrattle_effect`    | 死亡效果，例如返金币、补位、召唤                                                               |
| `on_ally_death_gain`    | 友方死亡成长规则和上限                                                                    |
| `on_kill_gain`          | 击杀成长规则和上限                                                                      |
| `field_count_scaling`   | 根据场上友方动物数量触发的光环或自增益                                                            |
| `attack_shape`          | single / cleave / line / cone / scatter / chain                                |
| `target_priority`       | normal / frontline / backline / ranged / support / high_hp / low_hp / building |
| `bonus_vs_hp_threshold` | 对高生命目标额外伤害或毒压血的触发阈值                                                            |
| `internal_counter_role` | 可选内部平衡标签，例如 anti_ranged_assault / anti_heavy / anti_light_aoe；不进卡面和战斗 UI |
| `counter_notes`         | 可选策划备注，仅用于调表和复盘，不作为玩家可见字段 |
| `synergy_tags`          | 光环、护盾、复制等配合标签                                                                  |

字段约束：

- `has_skill = false` 时，`trigger_timing` 和各类效果字段必须为空，只保留属性、召唤、射程、速度和目标偏好。
- `skill_complexity = single_trait` 时，只允许填写 1 个主要效果字段，避免一张卡同时有战吼、亡语、光环和成长。
- `primary_trait` 用于卡牌描述和图标展示，文本必须能压缩成一句话。
- `range_tier = ranged` 时，默认使用 `hp_budget_class = ranged_half`；`range_tier = ultra_ranged` 时，默认使用 `hp_budget_class = ultra_ranged_third`。
- `target_priority` 可以用于无技能动物；它是 AI 目标选择，不等于技能描述。
- 克制关系只在流派、反制模板和关键动物上维护；不要求每张动物卡强制填写克制/被克制标签，也不在游戏内显示“克制/被克制”文字。
- 轻型动物默认 `spawn_count = 2`；只有紫色轻型上限样板可以到 `spawn_count = 3`，首版不做 5-8 只大群。

## 11. 审阅问题与已确认结论

已确认结论已经同步到上文规则和定位表：

（1）8 格编组中 2 格固定给金矿和防御塔，动物主要按 6 格设计。
（2）小数值基准采用：无技能绿色动物 1 攻 / 3 血，绿色塔 1 攻 / 10 血，基地 2 攻 / 30 血，绿色营地 8 血。
（3）技能预算采用：绿色约 75%-85% 无技能、蓝色约 55%-70% 无技能、紫色约 20%-40% 无技能。
（4）远程同品质生命约为普通近战 1/2，超远程约为 1/3。
（5）不要求每张动物卡强制填写克制/被克制标签，克制关系也不在游戏中显示。
（6）首版绝大多数有技能动物只保留 1 个特性，不做双触发或复合技能。
（7）轻型一般召唤 2 只，紫色轻型最多 3 只，首版不做 5-8 只大群。
（8）60 张动物的定位可以根据当前规则整体重排。
