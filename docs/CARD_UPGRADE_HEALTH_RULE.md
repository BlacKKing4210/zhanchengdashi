# 远程与召唤动物升级生命规则

更新日期：2026-07-18

## 目标

解决低基础生命的远程动物与召唤型动物在低等级时，原有百分比成长被整数取整后看不出属性变化的问题。

## 规则

- 远程动物：基础攻击范围大于近战范围的动物。
- 召唤型动物：`skill_effect` 为 `summon` 的动物，即使它是近战，也适用该规则。
- 这些动物保留原有的统一升级倍率，并在等级 `2、4、6……` 时各额外获得 `+1` 生命。
- 对应累计额外生命为 `floor(等级 / 2)`：Lv.1 为 `+0`、Lv.2-3 为 `+1`、Lv.4-5 为 `+2`。
- 防御建筑、金矿及其他非动物卡不适用此保底生命成长。

## 可见性与验收

- 卡牌详情显示“远程/召唤：每2级+1生命”。
- 升至 Lv.2 后，低生命远程或召唤动物的生命数值必须立刻可见地增加至少 1 点。
- 卡牌统计、单人 AI、多人各玩家自己的卡组，以及 AI 镜像卡组均使用同一套 `CardRules.card_stats` 公式。
